# 大富豪ローカルルール完全調査レポート

大富豪のローカルルールを網羅的に調査した結果、**既に実装済みの24ルールを除き、約120種類以上の未実装バリエーション**を発見しました。カード効果系、革命系、勝敗条件系、縛り系、カード交換系に分類して報告します。

## まとめ

本調査により、大富豪のローカルルールは**公式ルールから大きく逸脱したものまで含めて120種類以上**存在することが判明しました。特に**4、6、9、K**に関連するルールが豊富で、地域や集団によって細かな運用差があります。

実装時には**複数ルールの相互作用**（例：革命中の8切り、ジョーカー代用時の効果発生有無、禁止上がりと都落ちの優先順位など）を明確に定義することが重要です。

情報源として特に充実していたのは、note.comのLoger氏による100種類以上のTierランク付き解説記事、Wikibooks、Enpedia、日本大富豪連盟公式ルールでした。

#### カード操作系
- [ ] ゲリラ兵 - 場のカードと同数字をより多く持つ時、手札から脇に出せる
- [ ] カタパルト - 場のカードと同数字を持つ時、場に追加で出せる
- [ ] スペード返し - 特殊効果発動時、同数字スペードで効果キャンセル
- [x] ジョーカー返し - ジョーカー1枚に対してもう1枚のジョーカーを重ねて出せる
- [ ] バナナアイス - 同色6枚で数字並びを作った時、宣言して捨てられる

- [x] 弱見せ - 9を出す、次プレイヤーの最弱カードを公開
